# MCP Telegram Mini-App/Bot

## Overview
This project creates a lightweight Telegram Mini-App/Bot that connects with a Python MCP server to let users deploy tokens, check community trusts, and view their portfolios from inside Telegram.

## Requirements
- Python 3.6+
- `python-telegram-bot` library
- `flask` library

## Setup Instructions

1. **Install Dependencies**
   ```bash
   pip install python-telegram-bot flask
   ```

2. **Run the MCP Server**
   ```bash
   python mcp_server.py
   ```

3. **Run the Telegram Bot**
   ```bash
   python bot.py
   ```

4. **Open the Telegram Webapp**
   - Open the `index.html` file in a web browser.
   - Click on the buttons to interact with the Telegram Bot.
